About

Blog Posting Platform

DEV@Deakin is a blog posting platform for Deakin students and researchers.

To start posting and reading the blogs, user will need to create an account.

Implemented a secure full-stack development.

Features

  • Real-time comment feature: Array Union implementation - Users will be able to post their questions or articles according to their email addresses. They will also be able to comment on other users’ posts with their names. Each user can post any number of comments on other users' posts
  • Real-time like feature: Similar to the comment feature, the user will be able to like other users’ postings. But, the user will only be able to like once for each posting. Therefore, no double likes
  • Notification feature: The user who posted the question/article will be notified if someone posted comments or likes on their posts. All of the notifications will be in real-time.
  • Help Centre (DEV@Deakin automated ChatBot): If the user needs help with DEV@Deakin, they can go to the help section. Deakin's automated help bot will help the user with the FAQ
  • Sign-out feature: Every user will be able to sign out from their account
